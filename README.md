# Student Proposal System

A full-stack web application where students can register, login, submit proposals, track proposal status, and receive feedback. Admin can review proposals, approve/reject them, view analytics, manage development progress, and maintain activity logs.

🌐 Live Demo: https://student-proposall-1.onrender.com

---

## 🚀 Features

- Student registration and login
- Admin and student role-based dashboard
- Submit proposals with priority and deadline
- Proposal status tracking
- AI score and suggestion system
- Admin approval/rejection system
- Search proposals
- Notifications system
- Activity logs
- Analytics dashboard using Chart.js
- Team management
- Task management board
- Progress tracking
- Dark/Light theme toggle

## ⚙️ How It Works

1. Users register as Student or Admin.
2. Users login using credentials.
3. Students submit proposals.
4. Proposals are evaluated automatically.
5. Admin reviews and updates proposal status.
6. Approved proposals move to development section.
7. Progress, tasks, comments, and team members can be managed.
